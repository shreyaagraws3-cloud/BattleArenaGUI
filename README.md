# BattleArena-GUI
A Java OOP + Swing turn-based battle simulator.

## Overview
BattleArena-GUI is a small Java game that demonstrates Object-Oriented Programming, inheritance, interfaces, method overriding, and a turn-based combat engine. The game includes a simple Java Swing GUI that allows the user to select two enemies and watch them battle in a text-based log.

This project is ideal for showcasing:
- Java fundamentals
- OOP architecture
- GUI development
- Game logic design

## Features

### Object-Oriented Design
- Enemy base class
- Zombie and Ogre subclasses
- Interfaces: IEnemy, IZombie, IOgre
- Encapsulation, inheritance, polymorphism

### Turn-Based Combat
- Alternating attack phases
- HP tracking
- Randomized special attacks
- Unique enemy IDs
- Battle log output

### GUI (Java Swing)
- Dropdown selection for enemies
- Start Battle button
- Scrollable battle log
- Simple and clean layout


## How to Run
1. Clone or download the project.
2. Open it in IntelliJ, VS Code, or Eclipse.
3. Run `Main.java`.
4. Choose two enemies from the dropdown menu.
5. Start the battle and watch the log update.
